# Appethaize — Demo Site

A premium single-page demo website for **Appethaize**, a Thai restaurant at
75 Baxter Street, New York (Chinatown, Manhattan).

Built by Moreta Studio as a bespoke demo.

## Stack

- [Next.js](https://nextjs.org) (App Router, TypeScript)
- [Tailwind CSS v4](https://tailwindcss.com)
- Variable fonts (Fraunces + Manrope) via `next/font/google`
- No external images — the design is purely typographic (gradients, SVG grain, motion)

## Highlights

- Mobile-first, fully responsive single page
- Scroll-reveal animations via IntersectionObserver, with `prefers-reduced-motion` support
- Accessible: semantic landmarks, skip link, focus styles, ARIA labels
- Static prerender — ships as pure static HTML

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```
